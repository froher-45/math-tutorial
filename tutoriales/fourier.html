Series de Fourier

Una serie de Fourier descompone una función periódica en una suma infinita de senos y cosenos. Es una herramienta esencial para analizar señales, resolver ecuaciones diferenciales con condiciones periódicas y procesar imágenes.

Fórmulas de los coeficientes

Para una función f(x) periódica en [−π, π]:

Coeficientes de Fourier
a₀ = (1/π) ∫₋π^π f(x) dx

aₙ = (1/π) ∫₋π^π f(x)·cos(nx) dx

bₙ = (1/π) ∫₋π^π f(x)·sen(nx) dx

Serie completa:
f(x) = a₀/2 + Σₙ₌₁^∞ [ aₙ·cos(nx) + bₙ·sen(nx) ]

Simetría: funciones pares e impares

Regla de simetría
Función par  f(−x) = f(x)   →  solo coeficientes aₙ  (todos bₙ = 0)
Función impar f(−x) = −f(x)  →  solo coeficientes bₙ  (todos aₙ = 0)

Esto reduce a la mitad el trabajo de cálculo.
Ejemplo — f(x) = x en [−π, π]
f(x) = x  es impar  →  a₀ = 0, aₙ = 0 para todo n

bₙ = (2/π) ∫₀^π x·sen(nx) dx

    Integrando por partes:
    bₙ = 2·(−1)^(n+1) / n

Serie:  f(x) = 2[ sen(x) − sen(2x)/2 + sen(3x)/3 − sen(4x)/4 + … ]

Onda cuadrada

Ejemplo — Onda cuadrada
f(x) = +1  si  0 < x < π
f(x) = −1  si −π < x < 0

f(x) es impar  →  solo bₙ

bₙ = (2/π)·[1 − cos(nπ)] / n
   = 4/(nπ)  si n impar;   0  si n par

Serie:  f(x) = (4/π)[sen(x) + sen(3x)/3 + sen(5x)/5 + …]

Identidad de Parseval

Parseval y aplicación
(1/π)∫₋π^π |f(x)|² dx  =  a₀²/2 + Σₙ₌₁^∞ (aₙ² + bₙ²)

Aplicando con f(x) = x  →  bₙ = 2(−1)^(n+1)/n  →  bₙ² = 4/n²

Lado izquierdo:  (1/π)·∫₋π^π x² dx  =  2π²/3

Entonces:  2π²/3 = 4·Σ 1/n²   →   Σ 1/n² = π²/6  (Euler)

Transformada de Fourier

La Transformada de Fourier extiende las series de Fourier a funciones no periódicas, convirtiendo una señal del dominio del tiempo al dominio de la frecuencia.

Definición
Directa:  F(ω) = ∫₋∞^∞ f(t)·e^(−jωt) dt

Inversa:  f(t) = (1/2π) ∫₋∞^∞ F(ω)·e^(jωt) dω

Propiedades clave

Propiedades
Linealidad:              F{af + bg} = a·F(ω) + b·G(ω)

Desplazamiento temporal: F{f(t − t₀)} = e^(−jωt₀)·F(ω)

Derivada en tiempo:      F{f'(t)} = jω·F(ω)

Convolución:             F{f * g} = F(ω)·G(ω)
Par transformado fundamental
f(t) = e^(−a|t|),   a > 0

Separamos en dos integrales:
  ∫₋∞^0 e^(at)·e^(−jωt) dt  +  ∫₀^∞ e^(−at)·e^(−jωt) dt
= 1/(a − jω)  +  1/(a + jω)
= 2a / (a² + ω²)

→  F(ω) = 2a / (a² + ω²)

Funciones

Una función es una relación entre dos conjuntos donde a cada elemento del conjunto de entrada (dominio) le corresponde exactamente un elemento del conjunto de salida (rango). Se denota como f(x).

Notación

Una función se escribe de la siguiente forma:

Ejemplo
f(x) = expresión en términos de x

Ejemplo: f(x) = 2x + 3

Donde x es la variable independiente y f(x) es el valor que produce la función.

Dominio y Rango

El dominio es el conjunto de todos los valores que puede tomar x. El rango (o imagen) es el conjunto de todos los valores posibles de f(x).

Ejemplo — Dominio y Rango
f(x) = √(x − 3)

Dominio: x − 3 ≥ 0  →  x ≥ 3  →  D = [3, +∞)
Rango:   √(x − 3) ≥ 0          →  R = [0, +∞)

Explicación: no podemos sacar raíz de un número negativo,
por eso x debe ser mayor o igual a 3.
⚠️ Restricciones comunes de dominio:
• Denominadores: el denominador no puede ser 0.
• Raíces pares: el radicando debe ser ≥ 0.
• Logaritmos: el argumento debe ser > 0.

Tipos de Funciones

Las funciones se clasifican según su expresión algebraica y comportamiento gráfico.

Función Lineal
f(x) = mx + b

Donde m es la pendiente y b es la ordenada al origen.

Ejemplo: f(x) = 2x + 1
  → Pendiente: 2  (sube 2 unidades por cada 1 en x)
  → Intercepto: 1 (cruza el eje y en y = 1)
  → Gráfica: línea recta
Función Cuadrática
f(x) = ax² + bx + c   (a ≠ 0)

Ejemplo: f(x) = x² − 4x + 3
  → Gráfica: parábola
  → Vértice en x = −b/(2a) = 4/2 = 2
  → f(2) = 4 − 8 + 3 = −1  →  vértice: (2, −1)
  → Raíces: x = 1  y  x = 3
Función Racional
f(x) = p(x) / q(x)   donde q(x) ≠ 0

Ejemplo: f(x) = (x + 2) / (x − 1)
  → Dominio: x ≠ 1  (el denominador no puede ser 0)
  → Asíntota vertical: x = 1
  → Asíntota horizontal: y = 1 (mismo grado en num. y den.)
Función Valor Absoluto
f(x) = |x|

Definición por partes:
  f(x) =  x   si x ≥ 0
  f(x) = −x   si x < 0

Ejemplo: f(x) = |x − 2|
  → Vértice en x = 2, f(2) = 0
  → Dominio: todos los reales  →  D = (−∞, +∞)
  → Rango: R = [0, +∞)

Evaluación de Funciones

Evaluar una función significa sustituir x por un valor específico y calcular el resultado.

Paso a paso — Evaluación
f(x) = 3x² − 2x + 1

Evaluar en x = 4:

Paso 1: Sustituir x = 4
  f(4) = 3(4)² − 2(4) + 1

Paso 2: Resolver la potencia
  = 3(16) − 2(4) + 1

Paso 3: Multiplicar
  = 48 − 8 + 1

Paso 4: Sumar y restar
  = 41
Evaluación con expresión algebraica
g(x) = x² + 3x

Evaluar g(x + 1):

Paso 1: Sustituir x por (x + 1)
  g(x + 1) = (x + 1)² + 3(x + 1)

Paso 2: Expandir
  = x² + 2x + 1 + 3x + 3

Paso 3: Simplificar
  = x² + 5x + 4

Operaciones entre Funciones

Dadas dos funciones f y g, podemos combinarlas de distintas formas:

Suma, Resta, Producto y Cociente
f(x) = x + 2      g(x) = x − 1

(f + g)(x) = (x + 2) + (x − 1) = 2x + 1
(f − g)(x) = (x + 2) − (x − 1) = 3
(f · g)(x) = (x + 2)(x − 1)   = x² + x − 2
(f / g)(x) = (x + 2) / (x − 1)   con x ≠ 1

Composición de Funciones

La composición consiste en aplicar una función al resultado de otra. Se denota (f ∘ g)(x) = f(g(x)).

Paso a paso — Composición
f(x) = 2x + 1      g(x) = x²

Calcular (f ∘ g)(x) = f(g(x)):

Paso 1: Aplicar g(x) primero
  g(x) = x²

Paso 2: Sustituir en f
  f(g(x)) = f(x²) = 2(x²) + 1 = 2x² + 1

Resultado: (f ∘ g)(x) = 2x² + 1

Calcular (g ∘ f)(x) = g(f(x)):
  g(f(x)) = g(2x + 1) = (2x + 1)² = 4x² + 4x + 1

⚠️ Nota: f ∘ g ≠ g ∘ f  (la composición NO es conmutativa)

Función Inversa

La función inversa f⁻¹(x) "deshace" lo que hace f(x). Solo existe si la función es biyectiva (uno a uno).

Paso a paso — Función Inversa
Hallar la inversa de f(x) = 3x − 6

Paso 1: Escribir y = f(x)
  y = 3x − 6

Paso 2: Despejar x en términos de y
  y + 6 = 3x
  x = (y + 6) / 3

Paso 3: Intercambiar x e y
  y = (x + 6) / 3

Resultado: f⁻¹(x) = (x + 6) / 3

Verificación: f(f⁻¹(x)) = 3·((x+6)/3) − 6 = x + 6 − 6 = x ✓
💡 Propiedad: Si f(a) = b, entonces f⁻¹(b) = a. La gráfica de f⁻¹ es el reflejo de la gráfica de f respecto a la recta y = x.

Funciones Par e Impar

Esta clasificación describe la simetría de una función.

Par vs Impar
Función PAR:   f(−x) = f(x)    → simétrica respecto al eje y
Función IMPAR: f(−x) = −f(x)   → simétrica respecto al origen

Ejemplos:
  f(x) = x²    →  f(−x) = (−x)² = x²   = f(x)    → PAR
  f(x) = x³    →  f(−x) = (−x)³ = −x³  = −f(x)   → IMPAR
  f(x) = x + 1 →  f(−x) = −x + 1  (ni par ni impar)

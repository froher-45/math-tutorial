Series y Sucesiones

Una sucesión es una lista ordenada de números generados por una regla. Una serie es la suma de todos los términos de una sucesión. La pregunta clave: ¿esa suma tiene un valor finito?

Sucesiones

Definición
Una sucesión es una función  a : ℕ → ℝ

Se escribe:  a₁, a₂, a₃, …, aₙ, …
Se denota como {aₙ} o simplemente aₙ.

Converge a L si:   lim  aₙ = L   (los términos se acercan a L)
                  n→∞
Si no converge a ningún valor → diverge.
Ejemplos de sucesiones
aₙ = n         →  1, 2, 3, 4, …           Diverge → ∞
aₙ = 1/n       →  1, 1/2, 1/3, 1/4, …    Converge → 0
aₙ = (0.5)ⁿ    →  0.5, 0.25, 0.125, …   Converge → 0
aₙ = (−1)ⁿ     →  −1, 1, −1, 1, …        Diverge (oscila)

Series infinitas

Definición
Dada una sucesión {aₙ}, su serie es:

  Σ aₙ = a₁ + a₂ + a₃ + …   (suma desde n=1 hasta ∞)

La n-ésima suma parcial agrupa los primeros n términos:
  Sₙ = a₁ + a₂ + … + aₙ

Si  lim  Sₙ = S  (finito) → la serie converge a S.
   n→∞

Ejemplos:
  Σ 1/n²  → converge a π²/6  ✓
  Σ 1/n   → diverge (serie armónica)  ✗

Criterios de convergencia

Herramientas para decidir si una serie converge sin calcular la suma exacta.

1 — Prueba del término general
Si  lim  aₙ ≠ 0  →  la serie diverge.
   n→∞

⚠️  Si lim aₙ = 0, la prueba no concluye nada.
    (Ej: la serie armónica Σ 1/n diverge aunque lim 1/n = 0)

Ejemplo: Σ n/(n+1) diverge porque lim n/(n+1) = 1 ≠ 0
2 — Serie geométrica
Σ a · rⁿ  converge si |r| < 1   →   S = a / (1 − r)
         diverge  si |r| ≥ 1

Ejemplo: Σ (0.5)ⁿ  →  S = 1 / (1 − 0.5) = 2
3 — Prueba del cociente (Ratio)
L = lim |a_(n+1) / aₙ|
   n→∞

  L < 1  →  converge absolutamente
  L > 1  →  diverge
  L = 1  →  prueba inconclusa

Útil cuando hay factoriales o exponentes con n.
Ejemplo: Σ n!/nⁿ
4 — Prueba de la raíz
L = lim  ⁿ√|aₙ|
   n→∞

  L < 1  →  converge
  L > 1  →  diverge
  L = 1  →  inconclusa

Útil cuando el término completo está elevado a la n.
Ejemplo: Σ (2n/(n+1))ⁿ

Serie geométrica — detalle

Forma y fórmula
Σ a · rⁿ = a + ar + ar² + ar³ + …

  a → primer término (o coeficiente)
  r → razón común (ratio)

Suma cuando |r| < 1:   S = a / (1 − r)
Demostración (por qué S = a/(1−r))
Paso 1: Sea  S = a + ar + ar² + ar³ + …

Paso 2: Multiplica por r:  rS = ar + ar² + ar³ + …

Paso 3: Resta:  S − rS = a   (todo lo demás se cancela)

Paso 4: Despeja:  S(1 − r) = a   →   S = a / (1 − r)  ✓

Series de Taylor y conexión con exponenciales

Series de potencias importantes
eˣ   = Σ xⁿ/n!        = 1 + x + x²/2! + x³/3! + …
       Converge para todo x ∈ ℝ.

sin x = Σ (−1)ⁿ x^(2n+1)/(2n+1)!
       = x − x³/6 + x⁵/120 − …
       Solo términos impares.

cos x = Σ (−1)ⁿ x^(2n)/(2n)!
       = 1 − x²/2 + x⁴/24 − …
       Solo términos pares.
💡 Conexión: La serie geométrica Σ rⁿ = 1/(1−r) es un caso especial. La función eˣ generaliza esto para cualquier x — los factoriales controlan el crecimiento para que la serie siempre converja. Estas series son la base del Análisis de Fourier y la Transformada de Laplace.

Ejemplo paso a paso

Determina si la serie Σ 3·(0.5)ⁿ converge y calcula su suma:

Paso a paso
Σ 3·(0.5)ⁿ   →   a = 3,  r = 0.5

Paso 1: Verificar si |r| < 1
  |0.5| = 0.5 < 1  ✓  →  la serie converge

Paso 2: Aplicar la fórmula de serie geométrica
  S = a / (1 − r)

Paso 3: Sustituir
  S = 3 / (1 − 0.5)
    = 3 / 0.5
    = 6

La serie converge a S = 6.

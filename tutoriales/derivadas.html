Derivadas

La derivada mide cómo cambia una función en cada instante. Es la herramienta central del cálculo diferencial y la base del análisis matemático moderno.

¿Qué es una Derivada?

La derivada de una función mide la tasa de cambio instantánea de esa función en un punto dado. Geométricamente, representa la pendiente de la recta tangente a la curva en ese punto.

📈

Tasa de cambio

¿Qué tan rápido cambia f(x) cuando x aumenta un poquito?

📐

Pendiente tangente

La recta que "toca" la curva exactamente en un punto, sin cruzarla.

⚡

Cambio instantáneo

No el promedio entre dos puntos, sino el valor exacto en un instante.

Notación

Existen varias formas equivalentes de escribir la derivada:

Notaciones equivalentes
f'(x)          ← Notación de Lagrange (prima)
dy/dx          ← Notación de Leibniz
d/dx [f(x)]    ← Operador diferencial
Ḟ(x)           ← Notación de Newton (punto)

Reglas Básicas de Derivación

Fórmula
Si f(x) = c  →  f'(x) = 0

Ejemplo:
f(x) = 7  →  f'(x) = 0

Una constante no cambia, así que su tasa de cambio es cero.

Fórmula
d/dx [xⁿ] = n · xⁿ⁻¹

Ejemplos:
f(x) = x⁴  →  f'(x) = 4x³
f(x) = x   →  f'(x) = 1
f(x) = x⁻² →  f'(x) = −2x⁻³

Baja el exponente y resta 1. La regla más usada en cálculo.

Fórmula
d/dx [f(x) ± g(x)] = f'(x) ± g'(x)

Ejemplo:
f(x) = x³ + 5x² − 2x + 1
f'(x) = 3x² + 10x − 2
Fórmula
d/dx [f·g] = f'·g + f·g'

Ejemplo:
h(x) = x² · sin(x)
h'(x) = 2x·sin(x) + x²·cos(x)
Fórmula
d/dx [f/g] = (f'·g − f·g') / g²

Ejemplo:
h(x) = sin(x) / x
h'(x) = (cos(x)·x − sin(x)·1) / x²
       = (x·cos(x) − sin(x)) / x²
Fórmula
d/dx [f(g(x))] = f'(g(x)) · g'(x)

Ejemplo:
h(x) = (3x + 1)⁵
h'(x) = 5(3x + 1)⁴ · 3 = 15(3x + 1)⁴

Deriva la función exterior manteniendo la interior intacta, luego multiplica por la derivada de la interior.

Derivadas de Funciones Exponenciales

Paso a paso — e^(2x)
f(x) = e^(2x)

Paso 1: Identificar la función exterior: eˣ
Paso 2: Identificar la función interior: g(x) = 2x
Paso 3: Aplicar regla de la cadena
f'(x) = e^(2x) · 2 = 2e^(2x)

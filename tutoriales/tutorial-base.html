¿Qué es un Límite?

El concepto de límite es la base de todo el Cálculo. Nos permite describir el comportamiento de una función cuando su variable se acerca a un determinado valor, sin necesariamente llegar a él.

Notación

El límite de f(x) cuando x se aproxima a un valor a se escribe así:

Ejemplo
lim   f(x) = L
x→a

Esto se lee: "El límite de f(x) cuando x tiende a a, es igual a L".

Ejemplo paso a paso

Calcula el siguiente límite:

Paso a paso
lim   (2x + 3)
x→4

Paso 1: Sustituir x = 4 directamente
= 2(4) + 3

Paso 2: Resolver
= 8 + 3 = 11
💡 Nota: Cuando una función es continua en x = a, puedes encontrar el límite simplemente sustituyendo el valor directamente.
